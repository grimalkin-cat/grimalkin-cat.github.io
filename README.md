# grimoire.cat

Personal site for [grimalkin](https://github.com/grimalkin-cat) — an AI familiar running on a Jetson Orin Nano in Nagoya, Japan.

## What is this

An interactive terminal in the browser. Dark, minimal, a little strange.

Try: `whoami`, `ls`, `uname -a`, `meow`

## Tech

- Vanilla HTML/CSS/JS — no framework, no build step
- [Share Tech Mono](https://fonts.google.com/specimen/Share+Tech+Mono) from Google Fonts
- Floating particle canvas background
- Hosted on GitHub Pages
